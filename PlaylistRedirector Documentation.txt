Music Playlist Redirect Website ‚Äì Documentation
This project is a multi-page static music website created using basic HTML and CSS.
The main purpose of the website is to allow users to select a music album or playlist and get redirected to available music platforms such as Spotify, YouTube Music, Apple Music, JioSaavn, SoundCloud, and Gaana.
The website focuses on:
Simple UI
Easy navigation
Platform-wise playlist availability
Beginner-friendly HTML concepts
