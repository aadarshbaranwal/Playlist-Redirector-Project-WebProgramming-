What We Have Used So Far

So far, this website has been created using basic HTML elements such as:
• Simple navigation links
• Image with clickable image mapping
• Basic text formatting using <b>, <i>, and <u>
• <marquee> tag for scrolling text
• Headings and paragraphs for clean structure
• Redirect links to different platforms


Technologies Used Till Now

We have used:
• HTML5 for structure
• Image Mapping for interactive playlists
• Basic Inline Styling (like colors)
• Local Images for homepage playlists


What Will Be Added Further

In the future, we will add:
• More playlist links for different music platforms
• Dedicated pages for each playlist
• Better page layout with more structured sections
• More information pages like Contact, FAQs, and Updates
• Option to check which platforms support a playlist


Future Goals

Our aim is to make the platform:
• More detailed
• More organized
• Faster to navigate
• Easier for users to find their favourite playlists


Stay Updated
